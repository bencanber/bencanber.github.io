插入排序

思想：每次将后续序列插入到一个排好序的序列中

时间复杂度：最坏O(n2)最好(On) 平均 (On2)

稳定性：稳定 每次都是在前面已拍好序的序列中找到合适的位置

private static void insertSort(int[] array) { 
    for (int i = 1; i < array.length; i++) {
           int tmp = array[i];
        int j = i;
            while (j > 0 && array[j - 1] > tmp) {
             array[j] = array[j - 1];
                j--;
        }
        array[j] = tmp;
       }
   }

希尔排序

思想：在插入排序的基础上，选取不同的间隔分别进行插入排序

时间复杂度：最坏O(n2)最好(On) 平均 (O:n*logn)

稳定性：不稳定，当相等的两个数分到不同组

 private static void shellSort(int[] array) {
    //确定插入间隔
    int len = array.length;
    int step = 1;
    while (step < len / 3) {
        step = step * 3 + 1;
    }
    //对不同的间隔分别进行插入排序
    while (step > 0) {
        for (int i = step; i < len; i++) {
            int tmp = array[i];
            int j = i;
            while (j >= step && array[j - step] > tmp) {
                array[j] = array[j - step];
                j -= step;
            }
            array[j] = tmp;
        }

        step = (step - 1) / 3;
    }
}

冒泡排序

思想：比较相邻两个数的值大的值向后移动

时间复杂度：最坏O(n2)最好(On) 平均 (On2)

稳定性:稳定

private static void bubbleSort(int[] array) {
    for (int i = 0; i < array.length - 1; i++) {
        boolean flag = true;
        for (int j = 0; j < array.length - i - 1; j++) {
            if (array[j] > array[j + 1]) {
                flag = false;
                int tmp = array[j];
                array[j] = array[j + 1];
                array[j + 1] = tmp;
            }
        }
        if (flag) {
            return;
        }
    }
}

快速排序

思想：选定一个基准，根据基准将元素分成两部分，左边的比基准小，右边的比基准大，基准就移动到了正确的位置

时间复杂度：最坏O(n2)最好(Onlogn) 平均 (Onlogn)

稳定性:不稳定

private static void quick(int[] array){
    quickSort(array,0,array.length-1);
}
private static void quickSort(int[] array,int left ,int right){
    if(left>right){
        return;
    }
    int partition = getPartition(array,left,right);
    quickSort(array,left,partition-1);
    quickSort(array,partition+1,right);
}
private static int getPartition(int[] array,int left,int right){
    int pivot = array[left];
    int i = left+1;
    int j = right;
    while(true){
        while(j>0 && array[j] > pivot){
            j--;
        }
        while(i<right && array[i] < pivot){
            i++;
        }
        if(i>=j){
            break;
        }else{
            swap(array,i,j);
        }
    }
    swap(array,left,j);
    return j;
}
private static void swap(int[] array,int a,int b){
    int tmp = array[a];
    array[a] = array[b];
    array[b] = tmp;
}

选择排序

思想；将第一个元素依次与后面的元素比较，经过一轮排序得到最小的值，循环进行

稳定性：不稳定

时间复杂度：最坏O(n2)最好(On) 平均 (On2)

private static void selectSort(int[] array){
    for(int i=0;i<array.length-1;i++){
        int min = i;
        for(int j=i+1;j<array.length;j++){
            if(array[min] > array[j]){
                min = j;
            }
        }
        swap(array,i,min);
    }
}

堆排序

思想；将数组排成大顶堆，然后将最后一个元素和array[0]交换，然后重新调整成大顶堆

稳定性：不稳定

时间复杂度：最坏O(Onlogn)最好(Onlogn) 平均 (Onlogn)

private static void heapSort(int[] array){
    int len = array.length;
    for(int i=len/2-1;i>=0;i--){
        heapAdjust(array,i,len);
    }
    for(int i=len-1;i>0;i--){
        int tmp = array[0];
        array[0] = array[i];
        array[i] = tmp;
        heapAdjust(array,0,i);
    }
}
private static void heapAdjust(int[] array,int i,int len){
    int tmp = array[i];
    int j = 2*i+1;
    while(j < len){
        if(j<len-1 && array[j+1] > array[j]){
            j++;
        }
        if(tmp >= array[j]){
            break;
        }
        array[i] = array[j];
        i = j;
        j = 2*j+1;
    }
    array[j] = tmp;
}

归并排序

思想；将数组分为两个有序序列然后进行归并 —— 分治递归

稳定性：稳定

时间复杂度：最坏O(Onlogn)最好(Onlogn) 平均 (Onlogn)

private static void mergeSort(int[] array){
    sort(array,0,array.length-1);
}

private static void sort(int[] array, int left, int right) {
    int mid = (left+right)/2;
    if(left<right){
        sort(array,left,mid);
        sort(array,mid+1,right);
        merge(array,left,mid,right);
    }

}

private static void merge(int[] array, int left, int mid, int right) {
    int[] tmp = new int[right-left+1];
    int z = 0;
    int i = left;
    int j = mid+1;
    while(i<=mid && j<=right){
        if(array[i] <= array[j]){
            tmp[z++] = array[i++];
        }
        if(array[i] > array[j]){
            tmp[z++] = array[j++];
        }
    }
    while(i<=mid){
        tmp[z++] = array[i++];
    }
    while(j<=right){
        tmp[z++] = array[j++];
    }
    for(int x=0;x<tmp.length;x++){
        array[x+left] = tmp[x];
    }

}
